# AI-Word Lexicon

**Cut on sight** (slop in technical writing): delve/delves/delving, showcasing, underscores, intricate, multifaceted,
nuanced, meticulous(ly), commendable, noteworthy, remarkable, tapestry ("rich tapestry", "a tapestry of"), testament ("a
testament to", "stands as a testament"), realm ("in the realm of"), kaleidoscope, symphony, beacon, cornerstone,
versatile, profound, fascinating, intriguing, ingenious, captivating, "plays a crucial/pivotal/vital role", "pivotal
moment", "navigating the landscape".

**Flag, cut only when decorative** (legit in engineering prose otherwise): leverage, robust, seamless, scalable,
potential, findings, insights, comprehensive, holistic, streamline, optimize, landscape, ecosystem, framework, crucial,
essential, key, vital.

**Copula avoidance** — restore plain "is": "serves as" / "acts as" / "functions as" / "stands as" / "represents" (when
it means is) → "is".
